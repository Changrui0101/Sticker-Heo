### Twikoo

Twikoo can be added to the json file：[Preview json](/twikoo.json)

### Valine

[Preview json](/valine.json)

### Waline
Add a configuration item in the emoji option：
```js
 emoji: [
    'https://cdn.jsdelivr.net/gh/zhheo/Sticker-Heo@latest/Sticker-100/',
]
```
